# Bill of Materials

Prototype BOM for OpenDrop.

Project by Jean-Sébastien Niel.
GitHub: https://github.com/jsniel

## Main components

- ESP32-C3 Mini
- DS3231 RTC module
- 2-channel relay module
- MT3608 boost converter
- AZ Delivery Battery Expansion Shield 18650
- DC irrigation valve
- LED + resistor
- Push buttons
- 3D printed enclosure
